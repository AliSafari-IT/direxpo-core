# Architecture

## Overview

This project follows a component-based architecture pattern.

## Directory Structure

```
src/
├── components/     # Reusable React components
├── styles/         # Global and component styles
├── utils/          # Utility functions and helpers
└── index.ts        # Application entry point
```

## Components

All components are located in `src/components/` and follow naming conventions.

## Styling

CSS files are organized by feature in `src/styles/`.

## Testing

Tests mirror the source structure and use `.test.ts` suffix.
