# Sample Project

This is a sample React TypeScript project used to demonstrate direxpo-core capabilities.

## Structure

- `src/components/` - React components
- `src/styles/` - CSS stylesheets
- `src/utils/` - Utility functions
- `tests/` - Test files
- `docs/` - Documentation

## Features

- Component-based architecture
- Type-safe TypeScript
- Utility helpers
- Comprehensive tests
